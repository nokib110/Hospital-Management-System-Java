## Project Overview
A Java-based application designed to manage hospital operations efficiently.
This system utilizes **OOP principles** (Inheritance, Polymorphism, Encapsulation) and **Java Swing** for the GUI.

## Features
* **Authentication:** Secure Admin Login.
* **Patient Management:** Add, Update, Search, and Delete patient records.
* **Doctor Management:** Manage doctor details and specializations.
* **Appointments:** Book appointments between doctors and patients.
* **Smart Billing:** Auto-calculates bills based on disease type (e.g., Heart surgery costs more than Flu).

## How to Run
1. Download the `.java` files.
2. Open in IntelliJ IDEA or Eclipse.
3. Run `HospitalGUI.java`.
